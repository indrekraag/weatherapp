# Weather app (GitHub Pages package)

This package is ready for GitHub Pages.

## Files
- `index.html` — the app
- `.nojekyll` — tells GitHub Pages to serve the site as a plain static site

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload both files from this folder to the repository root.
3. In GitHub: **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/** (root).
6. Save.
7. Wait a few minutes, then open the Pages URL shown by GitHub.

## Notes
- The app is a static site and does not need your Mac as a server once published.
- The entry file must be `index.html`.
